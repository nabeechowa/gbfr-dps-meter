## вңЁ What is it

A single native `.exe` that shows your whole team's **live DPS** in Granblue Fantasy: Relink co-op вҖ”
per-player, per-skill, with full player stats and equipped sigils.

It **only reads** damage events from the game. It never writes to the game, never modifies gameplay,
and never touches your save.

## рҹ”’ Privacy & trust

This is a meter, not a data collector. Everything below is verifiable from how the app behaves:

- вң… **Nothing is stored permanently** вҖ” chat auto-deletes (last 50 shown; a quiet room is wiped after 30 min)
- вң… **No IP, no HWID, no save data, no personal info** вҖ” only a display name you choose yourself
